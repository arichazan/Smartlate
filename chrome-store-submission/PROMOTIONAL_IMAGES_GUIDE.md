# Promotional Images & Screenshots Guide

## Required Images for Chrome Web Store

### 1. Screenshots (REQUIRED - at least 1, up to 5 recommended)

**Specifications:**
- Size: 1280x800 pixels OR 640x400 pixels (recommended: 1280x800)
- Format: PNG or JPEG
- Max file size: 2MB each

**What to Show:**
Take screenshots that demonstrate your extension's features:

1. **Screenshot 1: Extension in Action** (MOST IMPORTANT)
   - Show the right-click context menu with language options
   - Have some text selected on a webpage
   - Caption: "Select text and choose your language with one right-click"

2. **Screenshot 2: Options/Settings Page**
   - Show the options page with AI provider selection
   - Languages configured
   - Tone settings visible
   - Caption: "Easy setup - choose your AI provider and configure languages"

3. **Screenshot 3: Translation Result** (Optional)
   - Show before/after of text being translated
   - Or show the popup notification confirming text was copied
   - Caption: "Instant translation and rephrasing with your preferred tone"

4. **Screenshot 4: Popup Interface** (Optional)
   - Show the extension popup when clicking the toolbar icon
   - Caption: "Quick access to settings from your toolbar"

5. **Screenshot 5: Multi-language Support** (Optional)
   - Show the context menu with multiple languages
   - Caption: "Translate to any language you configure"

**How to Take Screenshots:**
1. Load your extension in Chrome
2. Open a webpage (like Wikipedia, news site, etc.)
3. Select some text
4. Right-click to show the menu
5. Use Chrome DevTools or a screenshot tool (Cmd+Shift+4 on Mac) to capture at 1280x800
6. Crop and edit as needed

### 2. Small Promotional Tile (REQUIRED)

**Specifications:**
- Size: 440x280 pixels (exactly)
- Format: PNG or JPEG
- Max file size: 1MB

**Design Suggestions:**
- Use your extension icon as the focal point
- Add the text "Smartlate" prominently
- Include tagline: "Smart AI Translation & Rephrasing"
- Use a clean, professional background
- Consider showing a visual metaphor (globe, languages, AI symbols)
- Color scheme: Match your extension icon colors

**Tools You Can Use:**
- Canva (free, has templates)
- Figma (free)
- Adobe Express (free tier)
- Photoshop/GIMP

---

### 3. Large Promotional Tile (OPTIONAL but RECOMMENDED)

**Specifications:**
- Size: 920x680 pixels (exactly)
- Format: PNG or JPEG
- Max file size: 1MB

**Design Suggestions:**
- Similar to small tile but with more detail
- Can include feature bullets:
  - "Translate to Any Language"
  - "AI-Powered Rephrasing"
  - "One-Click Operation"
- Show multiple language flags
- More prominent branding

---

### 4. Marquee Promotional Tile (OPTIONAL - for featured extensions)

**Specifications:**
- Size: 1400x560 pixels (exactly)
- Format: PNG or JPEG
- Max file size: 1MB

**Design Suggestions:**
- Horizontal banner style
- Left side: Extension icon/branding
- Right side: Key features and benefits
- Professional, eye-catching design
- This is used if Chrome features your extension

## Quick Start: Minimum Required Images

To submit your extension, you need AT MINIMUM:

1. ✅ **1 Screenshot** (1280x800)
2. ✅ **Small Promo Tile** (440x280)

That's it to get started! You can add more later.

## Design Tips

### Color Scheme
Based on your icon, consider using:
- Primary colors from your logo
- Clean white or light backgrounds
- Professional, modern look

### Typography
- Use clear, readable fonts
- Sans-serif fonts work best (Arial, Helvetica, Roboto)
- Make sure text is legible at thumbnail size

### Content Guidelines
- Don't include misleading imagery
- Don't use Google, Chrome, or competitor branding
- Keep it professional
- Focus on YOUR extension's features

## Example Screenshot Workflow

1. **Open Chrome** with your extension loaded
2. **Navigate to a demo page** (Wikipedia, news site, blog)
3. **Select interesting text** (a paragraph or sentence)
4. **Right-click** to show context menu
5. **Take screenshot** at 1280x800 resolution
6. **Edit/crop** if needed to focus on the important parts
7. **Add subtle annotations** if helpful (arrows, highlights)
8. **Save as PNG** for best quality
